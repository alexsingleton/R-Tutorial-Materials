Using R as a GIS

The following PDF provide a basic introduction to R and use as a GIS for making maps and basic spatial analysis. The content of these worksheets have been developed collaboratively by Alex Singleton, Chris Brunsdon and Nick Bearman. Feel free to adapt and share these materials.

PDF Tutorial Materials

  • Reworking and Recoding Data
  • Importing External Data
  • Attribute Joins
  • Basic Maps
  • Basic Spatial Analysis
  • Converting Coordinates
  • Common Error Messages

Code

The tutorials are designed to be self contained - with all data used have downloads directly linked from within them. The repository also contains the R markdown files and data so these could be adapted for your needs.
